Để chạy được paddleOCR trên GPU CÙNG VỚI ULTRALYTICS (nếu không chạy cùng ultralytics thì lên trang chủ paddleOCR cài phien bản GPU như bình thường: https://paddlepaddle.github.io/PaddleOCR/en/ppocr/quick_start.html#11-install-paddlepaddle)
Bước 1: Cài bản paddlepadlle-gpu phiên bản 2.4.2 với CUDA 11.7 (phiên bản này chỉ phù hợp với python 3.10):
python -m pip install paddlepaddle-gpu==2.4.2.post117 -f https://www.paddlepaddle.org.cn/whl/windows/mkl/avx/stable.html

Bước 2: Tải paddleOCR phiên bản mới nhất bằng câu lệnh
pip install "paddleocr>=2.0.1" # Recommend to use version 2.0.1+
